# @sst-puppeteer/cli

CLI for [sst-puppeteer](https://github.com/sst-puppeteer/sst-puppeteer). Ships the `sst-puppeteer` bin for utility commands.

To run the **MCP server**, use `sst-puppeteer-mcp` from `@sst-puppeteer/mcp` — they are peer frontends. This package does not depend on `@sst-puppeteer/mcp`.

**v0.1 commands:**

- `sst-puppeteer version` — print version (the only fully implemented command)

**Future:** `doctor`, `logs`, etc.
